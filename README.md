# Employee Data Cleaning (Pandas)

## Overview

A simple Python project that demonstrates how to clean and preprocess a messy employee dataset using *Pandas* and *NumPy*.

## Dataset

* messy_employee_dataset_1000_rows.csv
* Contains missing values, invalid ages, inconsistent date formats, duplicates, and inconsistent city names

## What This Project Does

* Handles missing and invalid values
* Converts age to numeric and fixes invalid ranges
* Cleans and converts join_date to datetime
* Extracts joining_year
* Normalizes city names
* Removes duplicate records

## Tools Used

* Python
* Pandas
* NumPy

## How to Run

1. Open the notebook
2. Update the dataset path if needed
3. Run all cells
